# Harshith AI — Command Center

An AI-powered course command center that lives inside Frappe LMS.

- Real-time stats from your LMS (courses, students, enrollments, revenue, quiz performance)
- Claude-powered tools: course outline generator, teaching assistant, student-risk insights, lesson drafter, announcement composer
- Served at `/ai` behind your LMS login (Course Creator / System Manager only)

Built for [harshith.io](https://www.harshith.io/).
